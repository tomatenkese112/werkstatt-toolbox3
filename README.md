# werkstatt-toolbox3
Die Ultimate Werkstatt-Toolbox ist eine schlanke, schnelle und übersichtliche Web-App (PWA), die speziell für den täglichen Einsatz in der Metallwerkstatt und Ausbildung entwickelt wurde. Sie vereint die wichtigsten Rechen- und Nachschlagewerkzeuge für Fertigungsmechaniker, Zerspanungsmechaniker, Konstruktionsmechaniker und Azubis an einem Ort.
